1. 斜体和粗体
	1. *斜体*或_斜体_
	2. **粗体**
	3. ***加粗斜体***
	4. ~~删除线~~
2. 分级标题
	1. # 一级标题  最大
	2. ## 二级标题
	3. ### 三级标题
	4. #### 四级标题
	5. ##### 五级标题
	6. ###### 六级标题
3. < 与 &  
	1.<会不显示，要写&lt  
	2.&要写&amp  
4. 如果你确实想要依赖 Markdown 来插入 <br /> 标签的话，在插入处先按入两个以上的空格然后回车。
